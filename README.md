# sgdfd-acw-client-master
Ejecutar
```
mvn clean install 
```
Para generar el instalador y los recursos necesarios en ```/target```.
En la carpeta ```/target/acw``` estan los archivos que debera colocar en la nube, se esta usando github releases en este caso.
Revisar la documentacion de getdown para mas informacion:
https://github.com/threerings/getdown
